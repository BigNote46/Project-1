# 💰 Smart Budget Planner

**Smart Budget Planner** คือเว็บแอปพลิเคชันสำหรับวางแผนการเงินและคำนวณงบประมาณรายเดือนแบบง่ายๆ ช่วยให้ผู้ใช้สามารถบริหารจัดการรายได้ แบ่งสัดส่วนเงินออม และบันทึกรายจ่ายได้อย่างเป็นระบบ เพื่อไม่ให้ใช้เงินเกินงบประมาณที่ตั้งไว้

---

## 🚀 ฟีเจอร์เด็ดของระบบ (Key Features)

* **Dashboard สรุปผล Real-time:** แสดงยอดรายได้ทั้งหมด, คำนวณเงินออมตามเป้าหมาย (20%) และงบประมาณคงเหลือที่ใช้ได้ (80%) ทันทีเมื่อกรอกข้อมูล
* **ระบบคำนวณงบประมาณหักลบอัตโนมัติ:** เมื่อมีการเพิ่มรายจ่าย ระบบจะนำไปหักลบกับงบประมาณที่ใช้ได้ พร้อมอัปเดตตัวเลขหน้าจอทันทีโดยไม่ต้องรีเฟรชหน้าเว็บ
* **ระบบแจ้งเตือนเมื่อใช้เงินเกินงบ (Budget Alert):** ตัวเลขงบประมาณคงเหลือจะเปลี่ยนเป็น **"สีแดง"** อัตโนมัติเมื่อค่าใช้จ่ายทั้งหมดเริ่มเกินงบประมาณที่ตั้งไว้
* **ระบบจำข้อมูล (Data Persistence):** ใช้ **LocalStorage** ในการจัดเก็บข้อมูล ทำให้ข้อมูลรายได้และรายการรายจ่ายยังคงอยู่ครบถ้วน แม้ว่าจะปิดหรือรีเฟรชหน้าเว็บไปแล้ว
* **รองรับทุกหน้าจอ (Responsive Design):** หน้าเว็บสามารถยืดหดและแสดงผลได้อย่างสวยงามทั้งบนคอมพิวเตอร์และโทรศัพท์มือถือ

---

## 🛠️ เครื่องมือที่ใช้พัฒนา (Tech Stack)

* **Frontend:** HTML5, Vanilla JavaScript (ES6)
* **CSS Framework:** [Bootstrap 5](https://getbootstrap.com/) (ช่วยจัด Layout และตกแต่ง UI ให้ทันสมัย)
* **Icon Library:** [Font Awesome 6](https://fontawesome.com/) (สำหรับไอคอนกราฟิกประดับหน้าเว็บ)
* **Storage:** LocalStorage (Web Storage API) สำหรับบันทึกข้อมูลฝั่ง Client-si
